## Installation on macOS

```
brew install glfw glew glm freeglut freetype sfml
```

Change Makefile with the appropriate freetype path.

## Libraries

* GLEW
* GLFW
* GLM
* libpng
* PNG texture loader (https://github.com/DavidEGrayson/ahrs-visualizer)
